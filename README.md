# AI-Powered Job Recruitment & Candidate Management System

## 📌 Project Overview

The AI-Powered Job Recruitment & Candidate Management System is developed using Salesforce CRM to automate and streamline the recruitment process. It enables recruiters to manage job openings, candidate applications, recruitment workflows, reports, and dashboards efficiently. The project also incorporates AI-based candidate scoring to support better hiring decisions.

---

## 🎯 Objectives

- Automate the recruitment process.
- Manage job openings and candidate applications.
- Improve hiring decisions using AI Candidate Score.
- Generate reports and dashboards for recruiters.
- Reduce manual effort through Salesforce automation.

---

## ✨ Features
